++ FUTURE OF VOICE CODE TEST - ROSE TREMLETT ++

23/06/2018

This project uses Node v8.9.4 and NPM v.5.6.0.

IMPORTANT: run npm install before running the code.

To run the code, enter either

> npm start

OR

>node app/index.js

in the terminal from the project root folder. A list of client IDs within 100km of the start location should be run up in the terminal.

To test the code with Mocha, enter

> npm test

into the terminal.
